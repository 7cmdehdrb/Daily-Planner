# 프로젝트 전체 설명

## 프로젝트 한 줄 요약

**하루결**은 Expo SDK 54와 React Native로 만든 로컬 우선(local-first) 일일 계획·기록·분석 앱입니다. 사용자는 하루 계획을 시간 블록으로 만들고, 실제 수행한 활동을 기록한 뒤, 계획과 실제 수행의 차이를 분석할 수 있습니다.

## 목표

이 프로젝트의 목표는 하루 24시간을 더 명확하게 계획하고 복기할 수 있도록 돕는 모바일 앱을 제공하는 것입니다.

주요 목표는 다음과 같습니다.

1. 날짜별 하루 계획을 시간 블록 단위로 작성합니다.
2. 반복되는 하루 루틴을 템플릿으로 저장하고 재사용합니다.
3. 계획된 활동에 대한 로컬 알림을 예약합니다.
4. 실제 수행한 활동을 실시간으로 시작·종료하며 기록합니다.
5. 계획 시간과 실제 기록 시간을 비교해 일치율, 계획 외 시간, 미기록 시간 등을 계산합니다.
6. 사용자가 OpenAI API 키를 직접 저장한 경우에만 AI 피드백을 생성합니다.

## 핵심 사용자 흐름

```text
앱 실행
  ↓
로컬 SQLite DB 초기화 및 기본 카테고리/템플릿 준비
  ↓
날짜 선택
  ↓
템플릿 또는 직접 입력으로 계획 작성
  ↓
계획 적용 및 알림 예약
  ↓
실제 활동 시작/종료 기록
  ↓
리뷰 화면에서 분석 계산
  ↓
선택적으로 OpenAI 기반 피드백 생성
```

## 주요 기능

### 1. 홈

홈 화면은 선택된 날짜의 계획 상태, 계획 블록 수, 분석 일치율, 현재 진행 중인 활동, 다음 일정, 수동 활동 시작 기능을 보여줍니다.

### 2. 계획 관리

계획 화면에서는 특정 날짜의 하루 계획을 생성하고 수정합니다. 계획 블록은 시작 시간, 종료 시간, 제목, 카테고리, 메모를 포함하며, 시간 충돌 검사를 거칩니다.

### 3. 실제 활동 기록

기록 화면과 홈 화면에서 계획된 블록을 시작하거나 계획에 없는 수동 활동을 시작할 수 있습니다. 새 활동을 시작할 때 이미 진행 중인 활동이 있으면 기존 활동은 종료됩니다.

### 4. 리뷰 및 분석

리뷰 기능은 완료된 활동 로그와 계획 블록을 비교합니다. 분석 결과에는 전체 계획 시간, 전체 기록 시간, 계획과 실제가 겹친 시간, 계획 일치율, 자기계발 시간, 계획 외 시간, 미기록 시간이 포함됩니다.

### 5. 템플릿

템플릿은 반복적으로 사용하는 하루 시간표입니다. 기본 템플릿으로 고등학생, 대학생, 직장인 평일 루틴이 제공되며, 사용자가 직접 템플릿을 추가·수정·복사·삭제할 수 있습니다.

### 6. 로컬 알림

계획 블록에 대해 시작 전 알림, 시작 알림, 종료 알림을 OS 로컬 알림으로 예약합니다. 서버나 백그라운드 JavaScript 실행에 의존하지 않고, 계획 적용 시점에 알림을 예약합니다.

### 7. AI 피드백

AI 피드백은 선택 기능입니다. 사용자가 OpenAI API 키를 등록한 경우에만 분석 요약을 OpenAI API로 전송하여 한국어 JSON 피드백을 요청합니다. API 키는 SQLite가 아니라 SecureStore에 저장됩니다.

## 기술 스택

| 영역 | 사용 기술 |
| --- | --- |
| 앱 프레임워크 | Expo SDK 54, React Native |
| 라우팅 | Expo Router |
| 언어 | TypeScript |
| 상태 관리 | Zustand |
| 로컬 데이터베이스 | expo-sqlite |
| 보안 저장소 | expo-secure-store |
| 로컬 알림 | expo-notifications |
| 날짜/시간 처리 | dayjs 및 자체 유틸리티 |
| 폼/검증 | react-hook-form, zod |
| AI 연동 | OpenAI Chat Completions API 직접 호출 |

## 데이터 저장 정책

이 앱은 별도 백엔드 서버를 사용하지 않습니다.

- 계획, 기록, 분석, 템플릿, 카테고리, AI 피드백 결과는 로컬 SQLite DB에 저장합니다.
- OpenAI API 키는 SecureStore에 저장합니다.
- API 키가 없으면 OpenAI 네트워크 호출을 수행하지 않습니다.
- 앱의 핵심 기능은 네트워크 없이도 동작하도록 설계되어 있습니다.

## 주요 데이터 모델

| 모델 | 설명 |
| --- | --- |
| `Category` | 계획·활동을 분류하는 카테고리입니다. 자기계발 여부를 포함합니다. |
| `Template` | 재사용 가능한 하루 루틴의 상위 정보입니다. |
| `TemplateBlock` | 템플릿에 포함된 시간 블록입니다. |
| `DailyPlan` | 특정 날짜의 계획 상태입니다. |
| `PlannedBlock` | 특정 날짜 계획에 포함된 실제 시간 블록입니다. |
| `ActivityLog` | 사용자가 실제로 수행한 활동 기록입니다. |
| `DailyAnalysis` | 하루 분석 결과입니다. |
| `AIFeedback` | OpenAI 응답과 입력 요약을 저장한 피드백 기록입니다. |
